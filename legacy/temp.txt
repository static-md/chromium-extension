Capture, annotate, and share screenshots instantly.

1. Click the extension icon to capture any page
2. Annotate with brush, arrows, rectangles, text, and more
3. Upload to static.md for instant sharing, or save as a collaborative markdown note
4. Get a clean link — no accounts, no ads, no nonsense

New in v1.13: Save as Note — turn any screenshot into an editable markdown note on static.md. Annotate, add context, and share with a single link.

Features:
- One-click full page capture
- 11 annotation tools: brush, line, arrow, rectangle, text, crop, color picker, undo/redo
- Upload to static.md for permanent image hosting
- Save as Note: create a markdown note with your screenshot embedded
- Right-click any image to edit with StaticShot
- No sign-up required, no tracking, no watermarks

Built by developers, for developers. Open source at github.com/static-md/chromium-extension
